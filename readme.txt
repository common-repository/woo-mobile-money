== Installation ==

<h4 style="box-sizing: inherit; margin: 0px 0px 1rem; padding: 0px; font-size: 0.8rem; border: none; color: #32373c; clear: both; font-family: &quot;Open Sans&quot;, sans-serif; line-height: 1.5; letter-spacing: 0.05rem; text-transform: uppercase;">
  AUTOMATIC INSTALLATION
</h4>

<ul style="box-sizing: inherit; margin: 0px 0px 1.5em 3em; padding: 0px; list-style-position: initial; list-style-image: initial; color: #32373c; font-family: &quot;Open Sans&quot;, sans-serif; font-size: 18px;">
  <li style="box-sizing: inherit; margin: 0px; padding: 0px;">
    Login to your WordPress Admin area
  </li>
  <li style="box-sizing: inherit; margin: 0px; padding: 0px;">
    Go to "Plugins > Add New" from the left hand menu
  </li>
  <li style="box-sizing: inherit; margin: 0px; padding: 0px;">
    In the search box type "Woocommerce Mobile Money Plugin"
  </li>
  <li style="box-sizing: inherit; margin: 0px; padding: 0px;">
    From the search result you will see "Woocommerce Mobile Money Plugin" click on "Install Now" to install the plugin
  </li>
  <li style="box-sizing: inherit; margin: 0px; padding: 0px;">
    A popup window will ask you to confirm your wish to install the Plugin.
  </li>
</ul>

<h4 style="box-sizing: inherit; margin: 2rem 0px 1rem; padding: 0px; font-size: 0.8rem; border: none; color: #32373c; clear: both; font-family: &quot;Open Sans&quot;, sans-serif; line-height: 1.5; letter-spacing: 0.05rem; text-transform: uppercase;">
  NOTE:
</h4>

<p style="box-sizing: inherit; margin: 1em 0px; padding: 0px; color: #32373c; font-family: &quot;Open Sans&quot;, sans-serif; font-size: 18px;">
  If this is the first time you've installed a WordPress Plugin, you may need to enter the FTP login credential information. If you've installed a Plugin before, it will still have the login information. This information is available through your web server host.
</p>

<ul style="box-sizing: inherit; margin: 0px 0px 1.5em 3em; padding: 0px; list-style-position: initial; list-style-image: initial; color: #32373c; font-family: &quot;Open Sans&quot;, sans-serif; font-size: 18px;">
  <li style="box-sizing: inherit; margin: 0px; padding: 0px;">
    Click "Proceed" to continue the installation. The resulting installation screen will list the installation as successful or note any problems during the install.
  </li>
  <li style="box-sizing: inherit; margin: 0px; padding: 0px;">
    If successful, click "Activate Plugin" to activate it.
  </li>
</ul>
